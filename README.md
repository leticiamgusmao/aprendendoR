# Aprendendo R para Economia
Este repositório serve como memória de alguns scripts que criei quando estava aprendendo econometria na Faculdade de Economia da UFF.
Também tem scripts para projetos com o Núcleo de Inovação, Meio Ambiente e Sustentabilidade (NIMAS-UFF) e de quando fui bolsista no IPEA. 
O objetivo é sempre ter a mão códigos "chaves" para algumas séries temporais que são recorrentes.
Se quiser contribuir, fique a vontade, mas não recomendo usar os scripts para nada além de consulta.
Bons estudos!
